# 🤖 EDA: AI Job Replacement Trends (2020–2026)

## Overview
Exploratory Data Analysis on a synthetic dataset of 15,000 records
examining AI-driven job displacement across 10 roles, 8 industries,
and 9 countries from 2020 to 2026.

## Dataset
| Property | Value |
|---|---|
| Source | Synthetic (Kaggle) |
| Rows | 15,000 (14,906 after cleaning) |
| Columns | 20 |
| Period | 2020 – 2026 |


## Technologies
- Python 3.x
- pandas, numpy
- matplotlib, seaborn
- scipy

# Open notebook
jupyter notebook notebook.ipynb

## Limitations
- Dataset is fully synthetic, distributions are artificially uniform
- Findings should not be generalized to real-world labor markets
- Balanced categories are a construction artifact, not a real trend
